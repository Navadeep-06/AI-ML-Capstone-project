## 📦 Requirements

Install the required libraries before running the notebook.

```bash
pip install pandas
pip install numpy
pip install matplotlib
pip install scikit-learn
pip install joblib
```

Or install everything at once:

```bash
pip install pandas numpy matplotlib scikit-learn joblib
```
